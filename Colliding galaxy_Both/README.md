### Galaxy



## How use
Start with all parameter's
```bash
cd Galaxy
make ThreeBody
```

Start with 3 Body
```bash
cd Galaxy
make ThreeBody
```

Start with  Sun Earth Moon system
```bash
cd Galaxy
make Sun-Earth-Moon
```

Start Collision
```bash
cd Galaxy
make Sun-Earth-Moon
```

Clean project
```bash
cd Galaxy
make clean
```
